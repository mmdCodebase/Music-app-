# The Phoenix Project

A Non-Commercial Music Player Made With Flutter.

## Why Phoenix?
* Beautiful Glassmorphic Design
* Lyrics
* A Flashlight-Visualizer
* Share Now-Playing
* Ad-Free
* Wallpapers From Artwork
And a bunch more. Why not check it yourself?
